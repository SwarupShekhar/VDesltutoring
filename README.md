# Englivo — Professional English Fluency Platform

Englivo is a premium, AI-powered platform designed to help professionals master English fluency through real-time practice and personalized coaching.

## 🚀 Overview

- **AI-Powered Learning:** Real-time speech analysis and feedback.
- **Expert Coaching:** Scheduled sessions with professional tutors.
- **Dynamic Content:** Component-based CMS for specialized landing pages.
- **Scalable Infrastructure:** Built on Next.js, LiveKit, and PostgreSQL.

## 🛠️ Development

To set up the project locally:

1.  **Install Dependencies:** `npm install`
2.  **Environment:** Configure your `.env` based on the provided template.
3.  **Database:** Run `npx prisma generate` and migrations.
4.  **Start:** `npm run dev`
